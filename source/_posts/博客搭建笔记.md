一、搭建方案：GitHub Pages + Hexo

二、基本流程

1.安装

```
npm install -g hexo-cli
```

2.初始化

```
hexo init wp-blog
cd my-blog
npm install
```

3.写文章

```
hexo new "博客搭建笔记"
```

4.本地预览

```
hexo clean
hexo s
```

5.推到github pages

```
hexo d
```
